# Brain Tumor Detection – CNN

This project trains a convolutional neural network (CNN) from scratch to classify brain MRI images into four categories:
- **glioma**
- **meningioma**
- **pituitary**
- **no tumor**

The dataset is from Kaggle: [Brain Tumor Classification (MRI)](https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri).
